List of Academic Projects

  1. Peer-to-Peer Multicasting inspired from Huffman Coding

    A mechanism similar to Huffman-tree is used to create a multi-cast streaming tree, where the root is a video server and leaves represent the peers. We aim at generating the shortest average path from the root to a leaf, with the relationship of the algorithm to the Huffman method providing the optimal tree.

  2. Emotion Recognition from Speech using K-NN Algorithm

    From the speech of the speaker, we tried to predict the current emotional state of the speaker. By observing how close they are to the sample data sets and performing a voting method after finding a certain finite number of sample elements to which the unknown speech signal is likely to resemble.
